# Project Rules for AI Agents

## Git Rules

- **NEVER push to GitHub** unless the user explicitly asks for it. Commit locally, but wait for explicit instruction before pushing to remote.
- Always commit with meaningful messages describing what changed and why.
- **NEVER** add "Co-Authored-By" lines mentioning any AI/LLM in commit messages.
