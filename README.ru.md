# Vocab Curve

Интерактивный инструмент анализа словарного запаса для изучающих языки. Визуализирует рост вокабуляра по мере чтения — кривая роста, частотное распределение и сравнение покрытия между текстами.

**Демо-данные включены** — приложение содержит предвычисленный анализ словарного запаса для 3 немецких текстов разного уровня сложности. Исходные тексты не включены.

## Что делает

Загрузите любой текст, разбитый на главы/разделы, и инструмент:

- **Отслеживает новые слова по разделам** — видно, как кривая «новых слов» выравнивается с каждой новой главой
- **Группирует словоформы по леммам** — `sagt/sage/sagen/gesagt` → `sagen`, `ging/gegangen/gehen` → `gehen`
- **Сравнивает тексты** — матрица покрытия, мостовые слова между уровнями, лестница вокабуляра
- **Поиск по словам** — найдите любое слово, все его формы и где они встречаются

## Демо-данные

В демо анализируются 3 немецких текста уровней B1 → B2 → C1:

| Текст | Уровень | Слова | Уникальные леммы | Разделы |
|-------|---------|-------|-------------------|---------|
| 82 темы B1 экзамена | B1 | 19 168 | 1 623 | 82 темы |
| Die Tribute von Panem | B2 | 100 249 | 7 523 | 27 глав |
| Drei Kameraden (Ремарк) | C1 | 127 611 | 9 152 | 28 глав |

**Лестница вокабуляра** — при чтении всех трёх текстов по порядку:

```
Темы B1:          1 623 леммы
+ Panem:         +6 667 новых → 8 290 всего
+ Drei Kameraden: +5 735 новых → 14 025 общий словарный запас
```

## Быстрый старт (демо)

```bash
cd web && npm install && npm run dev
```

Откроется на `http://localhost:5173` с предвычисленными демо-данными.

## Анализ своего текста

### Зависимости

```bash
npm install
cd web && npm install && cd ..
python3 -m venv .venv && source .venv/bin/activate && pip install simplemma
```

### 1. Подготовьте текст в формате markdown

```markdown
## Глава 1

Текст здесь...

## Глава 2

Ещё текст...
```

### 2. Создайте конфиг

```yaml
input:
  file: "./texts/my-book.md"
  language: de

structure:
  split_pattern: "^## "
  title_pattern: "^## (.+)"

analysis:
  stemmer: simplemma
  min_word_length: 2
  stop_words: true

output:
  path: "./output/my-book.json"
```

### 3. Запустите пайплайн

```bash
npx tsx src/cli.ts my-config.yaml
cp output/my-book.json web/public/
cd web && npm run dev
```

## Как это работает

```
Markdown → Токенизация → Лемматизатор simplemma → Частотный анализ → JSON → React-дашборд
```

Пайплайн извлекает текст из markdown, группирует словоформы по леммам с помощью [simplemma](https://github.com/adbar/simplemma) (словарный Python-лемматизатор, точность 0.95 на немецком), и вычисляет прогрессию словарного запаса по разделам. Дашборд читает предвычисленный JSON — никакого парсинга в рантайме.

## Визуализация

### Вид одного текста

- **Кривая вокабуляра** — столбцы (новые слова/раздел) + линия (накопительные уникальные)
- **Выравнивание кривой** — фильтр по порогу частотности: ядро vs редкие слова
- **Частотное распределение** — слова, встречающиеся 1×, 2×, 5×, 10×+
- **Обозреватель слов** — поиск, все словоформы, разделы с вхождениями

### Вид сравнения

- **Лестница** — каскадная диаграмма накопления вокабуляра по текстам
- **Матрица покрытия** — N×N сетка пересечений
- **Кривые вместе** — наложенные кривые роста словарного запаса
- **Мостовые слова** — самые частотные слова, которые нужно выучить для следующего уровня
- **Обзор текстов** — статистика и топ-слова по каждому тексту

## Технологии

- **Анализ**: TypeScript + Node.js + simplemma (Python)
- **Визуализация**: React + Vite + Recharts + Tailwind CSS v4
- **Данные**: предвычисленный JSON, без анализа в рантайме
- **i18n**: переключение интерфейса English / Русский
