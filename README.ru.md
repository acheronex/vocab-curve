# Vocab Curve

Интерактивный инструмент анализа словарного запаса для изучающих языки. Показывает кривую накопления словаря, визуализирует покрытие между текстами и экспортирует списки слов по частотным уровням для Anki.

## 🌐 Живая демо-версия

**Попробуйте сейчас:** [vocab-curve.erosheve.ru](https://vocab-curve.erosheve.ru/)

Исследуйте статистику словаря по 10 текстам немецкой литературы (~4.3 млн слов). Все функции визуализации работают в демо.

> **Примечание:** Экспорт в Anki отключён в демо. Клонируйте репозиторий для использования этой функции локально.

## Концепция

Накопление словарного запаса следует предсказуемой кривой — новый текст сначала вводит много новых слов, но по мере чтения всё меньше слов оказываются действительно новыми. Этот инструмент делает кривую видимой, помогает выбрать, что читать дальше, и показывает, какие именно слова нужно выучить.

## Визуализация (React-дашборд)

**Три основных режима:**

### 1. Сравнение (до 5 текстов)
- **Лестница** — каскадная диаграмма накопления словаря
- **Матрица покрытия** — N×N таблица, показывающая, сколько словаря одного текста покрывает другой
- **Статистика выбранных текстов** — агрегированные данные для выбранных текстов
- **Обзорные карточки** — статистика, плотность, покрытие ядра, топ-слова
- **Кривые словаря** — наложенные кривые роста для выбранных текстов
- **Слова-мосты** — конкретные слова, необходимые для перехода на следующий текст

### 2. Анализ одного текста
- **Сглаживание кривой** — фильтр по порогу частоты (Все / 2+ / 6+ / 20+)
- **Кривая словаря** — столбцы (новые слова/секция) + кумулятивная линия
- **Распределение частот** — горизонтальная диаграмма частотных уровней
- **Исследователь слов** — поиск любого слова, формы, секции, примеры
- **Метрики** — Секции, Слова, Токены, Уникальные леммы, Плотность, Покрытие ядра

### 3. Статистика корпуса (все тексты)
- **Обзор корпуса** — общие слова, уникальные леммы, процент пересечения словаря
- **Словарь по текстам** — горизонтальная диаграмма, отсортированная по леммам
- **Плотность словаря** — горизонтальная диаграмма, отсортированная по индексу Гиро
- **Рейтинг сложности** — тексты, отсортированные по нормализованной плотности
- **Частые слова корпуса** — топ-30 слов по всем текстам
- **Сортируемая таблица** — все тексты с метриками

Интерфейс поддерживает переключение **Английский / Русский**.

## Анализ своего текста

1. Отформатируйте как markdown с заголовками секций `## `.
2. Создайте конфиг-файл (скопируйте `config-lutherbibel.yaml` как шаблон).
3. Запустите: `npx tsx src/cli.ts my-config.yaml`

## Добавление текстов в сравнение

Добавление нового текста **не требует изменений в коде**:

1. Подготовьте markdown-файл в `for-analysis/`
2. Создайте конфиг YAML (включите поле `label`)
3. Запустите анализ: `npx tsx src/cli.ts config-new.yaml`
4. Запустите сравнение: `npx tsx src/compare.ts`
5. Скопируйте: `cp output/*.json web/public/`

## Экспорт в Anki

Экспортирует списки слов по частотным уровням как `.txt` файлы для импорта в Anki.

**Частотные уровни:**

| Уровень | Описание |
|---------|----------|
| 1× (гапакс) | Слова, встречающиеся только один раз |
| 2–5× (редкие) | Менее распространённая, но полезная лексика |
| 6–19× (средние) | Важные слова для беглости |
| 20+× (ядро) | Базовый словарь — обычно покрывает ~60% текста |

```bash
# Только статистика
python3 scripts/anki_export.py output/lutherbibel-1912.json --stats-only

# Экспорт всех уровней
python3 scripts/anki_export.py output/lutherbibel-1912.json

# Экспорт конкретного уровня
python3 scripts/anki_export.py output/lutherbibel-1912.json --tier 20+
```

## Как это работает

```
Markdown → Токенизация → simplemma лемматизатор → Частотный анализ → JSON → React-дашборд
                                                                              ↘ Anki TSV экспорт
```

[simplemma](https://github.com/adbar/simplemma) — словарный лемматизатор на Python (точность 0.95 на немецком). Правильно группирует словоформы — `sagt/sage/sagen/gesagt` → `sagen`, `wäre` → `sein`.

## Технологии

- **Пайплайн анализа**: TypeScript + Node.js + Python (simplemma)
- **Визуализация**: React 19 + Vite 8 + Recharts + Tailwind CSS v4
- **Экспорт Anki**: Python, выводит TSV
- **Данные**: предвычисленный JSON, без парсинга в рантайме
- **i18n**: переключение Английский / Русский
